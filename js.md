# next gen JS notes 

## Let and const
- Let has local scope and is used instead of var
- const is a constant whose cannot be changed 
- Let cannot be called before the initialisation unlike var

## Arrow functions
1. general syntax 
`
const f = (arg1, arg2) => {
	let a = 1
	return a
}
`
2. single argument syntax 
`
const f = arg1 => {
	let a = 1
	return a
}
`
